Flying Bananas


Create a sprite game with multiple flying objects -- bananas -- that you need to catch. In this game, we also added pedestrians, which lose points if they are hit by bananas. Both bananas and pedestrians need to go into arrays, where you can move them and check any collisions using a forEach loop.

You can "mod" this game in any way you like - change bananas to cupcakes or fireballs!
To see a working example click here


Directions
  1. If you are coding outside of WoofJS.com (like in Sublime), insert the Woof_JS source tag into the <head>section of your project page:

    <script src="https://rawgit.com/stevekrouse/WoofJS/master/dist/woof.js"></script>

    Your code now speaks Woof!

  2. Select a background appropriate for your game. Create a Catcher with an image you like.

  3. Have the Catcher "glide" using direction keys. This means that when you press a direction key, you don't change the Catcher's x or y position but rather their xVelocity or yVelocity.

    HINT: don't forget to add friction to slow them down a little bit over time.

  4. Make sure the catcher reacts properly when they hit an edge of the screen. They can pass thru to the other side of the screen, stop or bounce.

  5. Pick a Flying Object you want to fly at the Catcher. (apple, bowling ball?) Have one created Every second or two at the top or side of the screen, depending on if they are flying sideways or falling down. Remember to push each one into an array.

  6. Now, use a forEach loop to move every Flying Object. If the flying object hits the edge of the screen, have it be deleted from the screen and from the array.

  7. Check if one of the Flying Objects is hitting the Catcher. If so, remove it. Then add or subtract points, show a fun collision graphic and play a sound if you want.

  8. Add a score indicator on top of the screen. This can either be a text object, showing the number of points, or a "health bar" Rectangle object that changes its width with the score.

  9. Now add another set of moving sprites - Target Sprites. These can protect your Catcher or get hit if your Catchers fails to catch a Flying Object.

  10. If you want to have each Target Sprite look different, find different images and put them all into an Image Array. Then when you are creating a new Target Sprite, generate a random number, and assign the url: attribute to be one of the images in the Image Array, like: 
    
var targetSprite = new Image ({
     url: arrayOfImages[randomNumber],
     height:   etc 
})

  11. Test and finish the project till it looks perfect to you.
